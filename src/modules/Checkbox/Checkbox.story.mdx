# Checkbox

The base Checkbox is composed of a toggling Icon and a Text.

    <Story name="Checkbox">

To override the __hover__ effect you can customize it like this

```javascript
container: {
    "@selectors": {
      ":hover .y-checkbox-icon:not(.icon-disabled):not(.icon-checked):not(.icon-show-as-checked)": {
        // custom styles here
      }
    }
  },
```
