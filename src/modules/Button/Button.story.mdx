### Building a new Button

Extend the BaseButton by using the `extendStyles` property:

```javascript
const DerivedButton = BaseButton.extendStyles((theme) => ({
  button: {
    /* Derived Button default styling */
  },
  disabled: {
    /* Derived Button disabled styling */
  }
});
```
