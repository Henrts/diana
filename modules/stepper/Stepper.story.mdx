# Stepper

This element allows the user to track some progress as steps. It can be used just as read-only component, but can also allows the user to click and advance on some steps.0

### Available props

All the props are passed through to the `Stepper` element, so refer to its documentation for more customization options.

<table width="100%">
  <thead>
    <tr>
      <th>Prop Name</th>
      <th>Type</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>steps</td>
      <th>number</th>
      <td>Number of steps available</td>
    </tr>
    <tr>
      <td>activeStep (optional)</td>
      <th>number</th>
      <td>Current step active.</td>
    </tr>
    <tr>
      <td>iniitalActiveStep (optional)</td>
      <th>number</th>
      <td>Initial step active, default to 0</td>
    </tr>
    <tr>
      <td>clickable (optional)</td>
      <th>boolean </th>
      <td>Possible to change steps</td>
    </tr>
    <tr>
      <td>onChange (optional)</td>
      <th>(stepIndex) => void </th>
      <td>Callback when changing a step with the new step index</td>
    </tr>
  </tbody>
</table>
