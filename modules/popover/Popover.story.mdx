# Popover

The `Popover` is a component used to show and hide a modal element. It consists of an `Header` element,
which is used to toggle the element and show the current state when it is closed, and the `children` elements that are shown on the modal.

### Available props

<table width="100%">
  <thead>
    <tr>
      <th>Prop Name</th>
      <th>Type</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>centered (optional)</td>
      <th>boolean</th>
      <td>Whether the Popover should be centered with its anchor. Defaults to false.</td>
    </tr>
    <tr>
      <td>children</td>
      <th>ReactElement</th>
      <td>The content of the Popover.</td>
    </tr>
    <tr>
      <td>direction (optional)</td>
      <th>"bottom" | "left" | "right" | "top" | "bottom-right" | "top-right"</th>
      <td>The direction the modal window should take. Defaults to `bottom`.</td>
    </tr>
    <tr>
      <td>disabled (optional)</td>
      <th>boolean</th>
      <td>Whether the Popover should be opened or not. Defaults to false.</td>
    </tr>
    <tr>
      <td>dismissOnClick (optional)</td>
      <th>boolean</th>
      <td>
        Whether the Popover should close when there is a click outside of it. Defaults to true.
      </td>
    </tr>
    <tr>
      <td>renderHeader (optional)</td>
      <th>(visible: boolean) => React.ReactNode</th>
      <td>Renders the Popover header if it is passed.</td>
    </tr>
    <tr>
      <td>showOnHover (optional)</td>
      <th>boolean</th>
      <td>Whether the Popover should open on hover instead of on click. Defaults to false.</td>
    </tr>
    <tr>
      <td>useParentWidth (optional)</td>
      <th>boolean</th>
      <td>Whether the Popover should be as wide as its parent element. Defaults to false.</td>
    </tr>
    <tr>
      <td>onShow</td>
      <th>Function</th>
      <td>Called when the Popover content is shown.</td>
    </tr>
    <tr>
      <td>onHide</td>
      <th>Function</th>
      <td>Called when the Popover content is hidden.</td>
    </tr>
    <tr>
      <td>showOverlay (optional)</td>
      <th>boolean</th>
      <td>The popover will create a back overlay that will fill the page</td>
    </tr>
  </tbody>
</table>
