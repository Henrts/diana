# Checkbox

The base Checkbox is composed of a toggling Icon and a Text.

These are the custom props for the Base Checkbox

<table width="100%">
    <thead>
        <tr>
            <th>Prop</th>
            <th>Description</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>showAsChecked</td>
            <td>
                This props enables the possibility to show the checkbox as
                checked even though it is not checked specifically (i.e.
                checking the checkbox when "All options" is selected).
            </td>
        </tr>
        <tr>
            <td>IconChecked</td>
            <td>

                <Icon name="check" />`
            </td>

    </tbody>
</table>

To override the **hover** effect you can customize it like this

```javascript
container: {
    "@selectors": {
      ":hover .icon:not(.disabled):not(.checked):not(.show-as-checked)": {
        // custom styles here
      }
    }
  },
```
