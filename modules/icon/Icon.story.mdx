# Icon

### Prefered use

Use `Icon` to build a Icon Component. The prop `name` can be used to reference one of the icons. Please check all available icons in this section.

<Preview>
  <Story name="Base Icon">
    <Icon name="add" />
  </Story>
</Preview>

### Loading Custom Icons

You can also use the `src` prop to load a custom icon not in the system

<Preview>
  <Story name="Custom Icon">
    <Icon src="assets/icons/arrow.svg" />
  </Story>
</Preview>

### Icon Sizes

Use the `width` and `height` prop to define icon sizes.
The prop `size` changes both icon width and height.

<Preview>
  <Story name="Icon Sizes">
    <div
      style={{
        display: "flex",
        justifyContent: "space-around",
        alignItems: "center"
      }}
    >
      <Icon name="add" height={20} width={20} />
      <Icon name="add" height={30} width={30} />
      <Icon name="add" height={50} width={50} />
      <Icon name="add" size={50} />
    </div>
  </Story>
</Preview>

### Icon Colors

Both `stroke` and `fill` are available to change the icon colors. `color` changes both fill and stroke of the icon.

<Preview>
  <Story name="Icon Colors">
    <div
      style={{
        display: "flex",
        justifyContent: "space-around",
        alignItems: "center"
      }}
    >
      <div
        style={{
          display: "flex",
          flexDirection: "column",
          alignItems: "center"
        }}
      >
        <H3>Fill</H3>

      </div>
      <div
        style={{
          display: "flex",

          alignItems: "center"
        }}

      </div>
    </div>
  </Story>
</Preview>
